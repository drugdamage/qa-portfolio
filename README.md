# QA Portfolio

Учебное портфолио по тестированию ПО. Ведётся в процессе прохождения курса
Stepik «Тестирование ПО с нуля» — конспекты теории, задания по тест-дизайну,
SQL, работа с API через Postman и учебные проекты.

## Структура репозитория

- **`01-theory/`** — конспекты теории (SDLC, STLC, принципы тестирования и т.д.)
- **`02-test-design/`** — техники тест-дизайна, тест-кейсы, чек-листы
- **`03-sql/`** — задачи и запросы SQL для тестировщика
- **`04-api-postman/`** — тестирование API, коллекции Postman
- **`05-projects/`** — итоговые учебные проекты
